## Getting Started

First, run the development server (I prefer to use `pnpm` as package manager, but you can use `npm` or `yarn` as well):

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:1912](http://localhost:1912) with your browser to see the result.
